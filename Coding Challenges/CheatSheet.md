# Cheat Sheet:

- Finding min/max of something OR # of ways

  - Dependent - DP
  - Independent - Greedy (Local Optimal choices)
    - Prims
    - Kruskals
    - Dijkstra
  - Backtracking

- PrefixSum

  - Range Sum of elements multiple times within an array in a sliding window manner

- Array / String

  - 2 pointers
  - Sliding Window
  - Sorted Array:
    - Binary Search
    - 2 Pointers

- Finding specific element:

  - Use hashmap or set

- Monotonic Stack

  - finding the next greatest or smallest element in a sequence

- Stack:

  - matching nested structure
  - process elements in reverse order

- Heap:

  - continuously asking for finding the min/max element

- Graphs:

  - BFS / DFS
    - BFS for shortes or fewest steps
